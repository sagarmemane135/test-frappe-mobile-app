# PWA Icons

Place your app icons in this directory with the following sizes:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- badge-72x72.png (for notifications)

You can generate these from a single high-resolution icon using online tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Design Guidelines

- Use a simple, recognizable icon
- High contrast for visibility
- Avoid text in small sizes
- Square format (will be masked on some platforms)
